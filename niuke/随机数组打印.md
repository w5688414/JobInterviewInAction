# problem
给定一个长度为N且没有重复元素的数组arr和一个整数M，实现函数等概率随机打印arr中的M个数。


# codes
```
class RandomPrint {
public:
    vector<int> print(vector<int> arr, int N, int M) {
        // write code here
        vector<int> res;
        for(int i=0;i<M;i++){
            int p=rand()%(N-i);
            res.push_back(arr[p]);
            swap(arr[p],arr[N-i-1]);
        }
        return res;
    }
};
```

# analysis
随机选择一个数，加入res集合，然后把这个数跟N-i-1位置的元素交换，然后我们再0~N-i中再选择一个数，以此类推。
